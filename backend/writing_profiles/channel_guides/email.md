# Email Newsletter Guide

## Format
- Subject line: 5-8 words, curiosity-driven
- Preview text: Complement the subject line
- Body: 3-5 sections, scannable
- Clear single CTA

## Structure
1. **Opening**: Personal note or timely hook (2-3 sentences)
2. **Main insight**: The key takeaway this week
3. **Product update**: What's new in Bito (keep brief)
4. **Curated link**: One external resource worth reading
5. **CTA**: Try a feature, read a blog post, reply with thoughts

## Sending Times
- Best: Tuesday-Thursday, 10am EST
