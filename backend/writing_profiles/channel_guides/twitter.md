# Twitter/X Content Guide

## Format
- Single tweets: Max 280 characters, punchy and memorable
- Threads: 3-7 tweets, each self-contained but flowing
- Use threads for how-tos, stories, and data breakdowns

## Best Performing Patterns
1. **Quick insight**: One sharp observation in a tweet
2. **Thread breakdown**: Deep dive on a topic in 5-7 tweets
3. **Hot take + reasoning**: Controversial statement → evidence
4. **Tip of the day**: Practical dev productivity tips
5. **Engagement bait (good kind)**: Polls, "what do you think?" questions

## Posting Times
- Best: Weekdays 12-2pm EST
- Good: 8-9am EST
